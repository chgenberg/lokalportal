## Så sparar du som PDF

1. Öppna valfri `.html`-fil i Chrome/Safari
2. Tryck `Cmd+P` (Print)
3. Välj **"Spara som PDF"**
4. Ställ in:
   - Pappersformat: **A4 Liggande** (Landscape)
   - Marginaler: **Inga** (None)
   - Bakgrundsgrafik: **Aktiverad** ✓
5. Spara
